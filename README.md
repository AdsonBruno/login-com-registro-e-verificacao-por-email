"# Login com registro e verificação de usuário por email

Projeto back-end *Cadastro de usuário com autenticação por email*

### Descrição do projeto
> O projeto consiste em um sitema de login onde o usuário realiza o cadasto e 
> então informando um nome de usuário e o seu email. Após o cadastro, é enviar 
> um email de confirmação que ele irá receber para só assim poder
> ter seu cadastro autorizado. <br/>
> Detalhe: O email possuí um time de verificação, após esse tempo, será preciso
> reenviar o email
